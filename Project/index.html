Project

DIGITAL SLAMBOOK

The Digital Slambook is an online platform designed to modernize the traditional concept of a slam book. It allows users to create personalized pages where friends and acquaintances can share their thoughts, memories, and personal information in a fun and interactive way.

Key Features:


User Profiles
Interactive Entries
Responsive Design
Integration with Social Media

Technologies Used

Frontend:
Tailwind CSS, FontAwesome, responsive design.
Database:
To store user profiles, entries, and customization settings.
Authentication:
Secure login and registration system to protect user data.
